# Runzbio Texture Pack

## Features
- **Low Fire** - Reduced fire overlay for better visibility
- **Low Shield** - Smaller shield design
- **Small Totem Pop** - Compact totem of undying texture
- **Clean Water** - Clearer, more visible water

## Installation
1. Copy this folder to `%appdata%\.minecraft\resourcepacks` (Windows) or `~/Library/Application Support/minecraft/resourcepacks` (Mac)
2. Open Minecraft
3. Go to Options > Resource Packs
4. Enable "Runzbio Texture Pack"

## Notes
- This pack requires Minecraft 1.20+ (pack format 15)
- For OptiFine users: Enhanced fire and totem animations included
